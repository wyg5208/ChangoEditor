欢迎安装 Chango Editor！

Chango Editor 是一个功能强大的代码编辑器，提供类似 Sublime Text 的专业编辑体验。

主要特性：
• 智能语法高亮（支持20+编程语言）
• 多标签页管理
• 文件浏览器
• 强大的查找替换功能
• 多主题支持（暗色/亮色主题）
• 自定义图标和界面

系统要求：
• Windows 7 SP1 或更高版本（推荐 Windows 10/11）
• 64位操作系统
• 至少 100MB 可用磁盘空间
• 1GB RAM（推荐 2GB 或更多）

安装说明：
1. 选择安装目录（默认：C:\Program Files\Chango Editor）
2. 选择是否创建桌面快捷方式
3. 选择是否关联常用代码文件格式
4. 点击"安装"开始安装过程

注意事项：
• 如果检测到旧版本，将自动卸载并安装新版本
• 安装过程中请不要运行其他程序
• 建议在安装前关闭所有正在运行的代码编辑器

感谢您选择 Chango Editor！
