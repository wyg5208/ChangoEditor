✨功能概览

Chango Editor 是一个基于 Python 和 PyQt6 开发的现代化代码编辑器，提供了类似 Sublime Text 的优雅编程体验。

🎨 智能语法高亮

支持 20+ 种编程语言的语法高亮，包括 Python、JavaScript、TypeScript、Java、C++、C#、Go、Rust 等主流语言。

📁 树形文件浏览器

直观的文件夹结构展示，支持懒加载、右键菜单、一键展开/收起等高级功能。

🖱️ 文件拖拽支持

直接从文件管理器拖拽文件到编辑器打开，支持批量操作和多种文件格式。

📋 多标签页管理

同时编辑多个文件，支持标签页拖拽排序、右键菜单、智能文件名显示。

🎯 图标化工具栏

现代化的图标界面，包含新建、打开、保存、编辑等常用功能，支持快捷键提示。

🌓 主题系统

深色/明亮主题无缝切换，护眼设计，所有界面元素完美适配。

📁文件管理

文件浏览器操作

  • 展开文件夹：点击文件夹前的展开箭头，或双击文件夹名称
  • 打开文件：双击文件名，会在新标签页中打开
  • 右键菜单：右键点击文件或文件夹，可进行更多操作
  • 导航按钮：
    • ⬆️ 上级目录：返回上一级目录
    • 🔄 刷新：手动刷新文件列表
    • ⬇️ 展开全部：一键展开所有文件夹
    • ⬆️ 收起全部：一键收起所有文件夹

文件拖拽功能

  • 从文件管理器直接拖拽文件到编辑器打开
  • 支持单个或多个文件同时拖拽
  • 拖拽时显示蓝色边框提示
  • 支持的文件格式：.py, .js, .ts, .html, .css, .cpp, .java, .cs, .php, .rb, .go, .rs, .json, .xml, .md, .txt 等

新建文件模板

使用 Ctrl+N 新建文件时，可以选择预定义模板：

  • Python 模板：包含标准文件头和主函数结构
  • JavaScript 模板：现代 ES6+ 语法模板
  • HTML 模板：标准 HTML5 页面结构
  • CSS 模板：基础样式表模板
  • 其他语言模板：TypeScript, Java, C++, C#, Go, Rust 等

✏️编辑功能

语法高亮

支持 20+ 种编程语言的专业语法高亮：

🐍 Python

完整支持 Python 3.x 语法，包括类型注解、装饰器、异步编程等现代特性

🌐 Web 技术

JavaScript、TypeScript、HTML、CSS，全面支持现代 Web 开发

⚙️ 系统语言

C、C++、Rust、Go 等系统编程语言的完整语法高亮

☕ 企业语言

Java、C#、PHP 等企业级开发语言的专业支持

编辑器特性

  • 行号显示：智能行号区域，自动调整宽度
  • 当前行高亮：清晰标识编辑位置
  • 撤销/重做：完整的操作历史管理
  • 智能缩进：根据语言特性自动缩进
  • 括号匹配：自动匹配和高亮显示括号

查找和替换

  • 使用 Ctrl+F 打开查找对话框
  • 使用 Ctrl+H 打开查找替换对话框
  • 支持正则表达式搜索
  • 大小写敏感选项
  • 全词匹配选项

🎨主题系统

主题切换

通过菜单栏 → 主题 可以切换编辑器主题：

  • 深色主题：专业的暗色配色方案，护眼设计，适合长时间编程
  • 明亮主题：清爽的亮色配色方案，适合白天使用

主题特色

  • 完全适配：所有界面元素都会根据主题变化
  • 即时生效：主题切换立即生效，无需重启
  • 一致性设计：工具栏、对话框、菜单等保持统一风格
  • 护眼优化：合理的颜色对比度，减少视觉疲劳

💡使用技巧

🚀 提高效率的小技巧

  • 使用 Ctrl+W 快速关闭当前文件
  • 拖拽多个文件到编辑器可以批量打开
  • 右键点击标签页可以快速关闭其他文件
  • 使用文件浏览器的右键菜单可以快速设置根目录

🎯 界面操作技巧

  • 鼠标悬停在工具栏按钮上查看功能说明
  • 使用展开/收起全部按钮快速管理文件夹视图
  • 调整左侧文件浏览器的宽度适应项目需求
  • 长文件名会自动省略显示，悬停查看完整路径

⚙️ 系统要求和性能

  • Python 版本：Python 3.8 或更高版本
  • 依赖库：PyQt6 6.9+, Pygments（语法高亮）
  • 操作系统：Windows 10/11，macOS，Linux
  • 内存占用：约 50-100MB，根据打开文件数量变化
